# Contributing

Thank you for considering contributing to Laravel Integrations!

## Development Setup

```bash
git clone https://github.com/vimatech-io/laravel-integrations.git
cd laravel-integrations
composer install
```

## Running Tests

```bash
composer test
```

## Code Style

```bash
composer format
```

## Static Analysis

```bash
composer analyse
```

## Pull Requests

- Fork the repo, create a feature branch
- Add tests for new features
- Ensure all tests pass and PHPStan is clean
- Submit a PR against `main`

## Security Vulnerabilities

Please report security issues to hello@adelzemzemi.dev directly.
